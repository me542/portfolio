AquaMansi: Device

Is an Arduino-based automated watering system designed to efficiently manage plant care. It uses soil moisture sensors to detect when plants need water, a real-time clock (RTC) to schedule irrigation, an SD card module to log data, and an LCD display to show current system status. A servo motor controls the water flow, while an ESP module enables wireless communication with the app. This device ensures that plants are only watered when necessary, promoting healthy growth and reducing water waste

AquaMansi: App

A mobile application that provides users with real-time access to their system. Through the app, users can monitor soil moisture levels, view watering history, receive alerts, manually activate the irrigation system, and configure settings remotely. Sweep

A Commission is an IoT-based monitoring system designed to track both waste level and water level in real time. Built with smart sensors and microcontrollers, the device automatically detects when a trash bin is full or when a water container reaches a certain level. The data is transmitted wirelessly, allowing users to receive alerts, monitor status remotely, and make timely decisions whether for efficient waste collection or water resource management.
